Arten der Unmöglichkeit im Schuldrecht

Definition: Die Leistung kann von niemandem mehr erbracht werden.

Beispiel: Ein bestimmtes Kunstwerk wurde zerstört und kann daher von keinem anderen Künstler nachgebildet werden.

Definition: Die Leistung kann vom Schuldner persönlich nicht erbracht werden, aber möglicherweise von einem Dritten.

Beispiel: Ein Pianist kann wegen Krankheit ein versprochenes Konzert nicht spielen, aber ein anderer Pianist könnte das Konzert übernehmen.

Definition: Die Unmöglichkeit besteht bereits vor Vertragsschluss.

Rechtsfolge: Der Vertrag bleibt wirksam, aber der Schuldner wird von seiner Leistungspflicht befreit.

Beispiel: Ein bereits gestohlenes Auto wird verkauft.

Definition: Die Unmöglichkeit tritt erst nach Vertragsschluss ein.

Rechtsfolge: Der Schuldner wird von seiner Leistungspflicht befreit, aber möglicherweise Schadensersatzpflicht.

Beispiel: Ein Auto wird nach dem Verkauf gestohlen.

Definition: Die Leistung muss zu einem bestimmten Zeitpunkt erbracht werden, sonst ist sie unmöglich.

Beispiel: Ein Konzert, das zu einem festgelegten Termin stattfinden sollte.

Gattungsschuld: Unmöglichkeit tritt erst ein, wenn die gesamte Gattung untergeht.

Stückschuld: Unmöglichkeit tritt ein, wenn das konkrete Stück untergeht.

Beispiel: Verkauf einer bestimmten Flasche Wein im Gegensatz zu einer Flasche aus einer Serie.

Definition: Die Leistung ist theoretisch möglich, aber der Aufwand ist unzumutbar.

Rechtsfolge: Der Schuldner kann die Leistung verweigern.

Beispiel: Ein wertvoller Ring fällt in einen tiefen See.

Definition: Die Erbringung der Leistung ist für den Schuldner persönlich unzumutbar, z.B. aus moralischen oder ethischen Gründen.

Beispiel: Ein Arzt weigert sich, eine lebensgefährliche Operation durchzuführen.

Gründe für den Eintritt der Unmöglichkeit

Beispiel: Das verkaufte Auto wird gestohlen oder das gemietete Haus brennt ab.

Beispiel: Ein Künstler, der eine bestimmte Performance erbringen sollte, verstirbt oder erkrankt schwer.

Beispiel: Alle Einheiten einer verkauften Produktserie werden zerstört.

Beispiel: Ein Lieferant kann eine Ware nicht liefern, weil ein Exportverbot verhängt wurde.

Beispiel: Ein Catering-Service liefert das Essen nach der Hochzeit, wodurch die Leistung ihren Zweck verfehlt.
